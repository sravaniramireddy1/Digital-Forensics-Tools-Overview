# Digital-Forensics-Tools-Overview
ðŸ”¹ Aim

To recover deleted or damaged files/partitions from a storage device using TestDisk.

ðŸ”¹ Description

TestDisk is an open-source data recovery utility. It helps to recover lost partitions and repair corrupted ones by analyzing the disk structure and restoring damaged boot sectors.

ðŸ”¹ Steps

Log Creation

Select Create to generate a log file.

Select None if logs are not needed.

Press Enter to proceed.

Disk Selection

Choose the target storage device from the list.

Press Enter to proceed.

Partition Table Selection

TestDisk auto-detects partition type.

Select the suggested type and press Enter.

<img width="1920" height="1080" alt="Screenshot 2025-08-31 214948" src="https://github.com/user-attachments/assets/0ebf3c70-d76d-45b5-a922-d27bee672b54" />

Analyse Partition Structure

Choose Analyse to check partition structure.

Use Quick Search to scan for lost partitions.

Press p to list files in detected partitions.
